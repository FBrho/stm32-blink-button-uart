# STM32 Blink + Bouton + UART

## Matériel
- STM32 Nucleo-F446RE
- Bouton intégré B1 (PC13)
- LED intégrée LD2 (PA5)

## Fonctionnalités
- Clignotement LED à plusieurs vitesses
- Changement de vitesse via bouton B1
- Affichage de la vitesse sur UART

## Compilation
- STM32CubeIDE
- Fichier `.ioc` inclus
